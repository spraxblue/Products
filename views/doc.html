1. Descrierea generala a proiectului

Ivy este o aplicatie web care pune la dispozitie utilizatorului o gama larga de produse si sfaturi de utilizare pentru Skin care & Makeup. Utilizatorii pot afla firme de produse Makeup si noi trenduri, dar si ce produse este recomandat sa foloseasca in functie de tenul si dorintele lor.

Aplicatia este destinata pentru oricine cu un minim interes fata de aceste subiecte, astfel chiar si cei la inceput de drum sa se poata bucura la fel de mult de aplicatie ca si cineva cu ani de experienta in domeniul de beauty.

2. Front-End

Interfata grafica este sub forma unei pagini web in care s-a folosit HTML5, CSS3 si Javascript. Acestea sunt dispuse in mai multe pagini responsive, care pot fi accesate dintr-o bara de navigare aflata in partea de sus a fiecarei pagini care poate varia in functie de tipul de utilizator: user sau admin.

2.1. Home Page

Home page integreaza 3 pagini web:

  • Landing page
  • About Us
  • Contact

Landing page - este pagina de bun venit si contine logo-ul si motto-ul acestei aplicatii.

About Us - contine descrierea misiunii, a comunitatii si scopul nostru.

Contact - este un formular de contact pentru doritori de detalii, feedback constructiv sau pentru propuneri smart.

In prim plan pagina are ca scop promovarea self awareness, self love chiar si cand vine vorba de ingrijirea tenului. UI&UX sunt cat mai atractive, tocmai deoarece este o aplicatie web de frumusete si ingrijire si astfel ne promovam scopul chiar din aspectul site-ului nostru.

2.2. Search Page

Aceasta poate fi accesata din bara de navigare aflata pe toate celelalte pagini si are ca scop fluidizarea informatiilor.

La acesarea search page-ului sunt afisate toate produsele existente pe site. Ele pot fi filtrate dupa bunul plac al utilizatorului cu ajutorul meniului de sortare din partea stanga a ecranului.

2.3. Skin care quiz Page

Formularul pentru skin care are ca rol oferirea de indicatii in legatura cu diversele tipuri de ten. Rezultatul quiz-ului vor fi 2 liste de ingrediente: una cu ingredientele recomandate si una cu ingredientele de evitat.

2.4. Login / Sign-up Page

Pentru a salva preferintele utilizatorului sau raspunsul formularelor, cat si history-ul acestuia, avem pagina de login pentru a se conecta in aplicatie. Daca acesta nu are cont exista butonul de register care ii va afisa un formular unde trebuie sa completeze datele personale. Cand acesta va apasa butonul de "Login" va fi redirectionat catre pagina "My Account".

2.5. My account Page

Pagina "My Account" afiseaza datele trecute in formularul de sign-up, iar utilizatorul poate sa isi modifice setarile contului (pentru care va avea in mod evidnt nevoie de parola contului sau). Pentru a schimba parola utilizatorul are butonul "Change password" unde va trebui sa confirme parola nou cat si cea veche pentru actualizare.

2.6. Admin Page

Admin page are mai multe functionalitati specifice administratorului aplicatiei:

  • adaugarea de produse noi
  • eliminarea unor produse existente
  • modificarea produselor existente

2.7. Responsive

Paginile incluse in aplicatia web sunt responsive (pana la dimensiunea de 280 pixeli - dimensiunea celui mai ingust telefon) cu ajutorul unor serii de media queries in CSS3.
